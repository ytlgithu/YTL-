{# 简介 #}

💡 简介

YTL-Warehouse-Blog 是一个个人技术平台，用于分享编程知识、存储技术文档与管理固件仓库。

在这里你可以记录学习笔记、发布技术文章、管理代码仓库，所有内容支持 Markdown 格式编写。

{# 技术栈 #}

🛠️ 技术栈

🐍 Flask 🗄️ PostgreSQL 💾 SQLAlchemy 🎨 Highlight.js 🖥️ Monaco Editor 🚀 Railway ⚡ Gunicorn 🐳 Docker
{# 功能 #}

✨ 功能

  • 📦 仓库管理 — 创建私有/公开仓库，上传固件、代码、文档，支持文件树浏览与 Markdown 渲染
  • 📝 博客系统 — Markdown 写文章，代码语法高亮，文章分类，公开/私有控制
  • 👤 用户系统 — 注册登录、个人头像、角色权限
  • 📊 操作日志 — 记录所有关键操作，方便追溯
  • 🔒 隐私控制 — 仓库与文章均可独立设置公开或私有
